Detailed Description

Driver library for the Si4703 FM Radio Tuner.

#include <si4703.h>
si4703.h

This library handles I2C communication with the Si4703 chip, allowing for frequency setting, volume control, seeking, and RDS reading.

void si4703_read_rds ( char * buffer,
uint16_t timeout_ms )

Read RDS (Radio Data System) data.

  • Attempts to read the Station Name (PS) from the RDS data stream. This function blocks for up to timeout_ms while waiting for valid data.
  • Parameters
    bufferPointer to a char array (min size 9) to store the station name.
    timeout_msMaximum time to wait for RDS data in milliseconds.
